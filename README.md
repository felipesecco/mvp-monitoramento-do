# 📄 MVP - Monitoramento Inteligente de Diários Oficiais

Este projeto apresenta um **MVP funcional** para monitoramento, estruturação e visualização de eventos extraídos de Diários Oficiais.

A solução demonstra, de forma prática, a capacidade de transformar conteúdo não estruturado em **informação organizada e consultável**, permitindo análises iniciais e validação de valor.

---

## 🚀 Demonstração

🔗 **Acesse o app:**  
https://SEU-APP.streamlit.app

---

## 🎯 Objetivo do MVP

- Validar a viabilidade técnica da solução  
- Demonstrar a estruturação de dados a partir de conteúdos não estruturados  
- Permitir visualização e filtragem de eventos relevantes  
- Servir como base para evolução do produto  

---

## ⚙️ Funcionalidades

- 📊 Visualização de eventos estruturados  
- 🔎 Filtros por:
  - tipo de evento  
  - órgão  
  - data de publicação  
- 📈 Distribuição por tipo de evento  
- 🏛️ Distribuição por órgão  
- 📋 Visualização detalhada dos registros  

---

## 🧠 Tipos de eventos monitorados

- Aposentadoria  
- Exoneração  
- Nomeação  
- Posse  
- Afastamento  
- Concurso  
- Promoção  
- Indicação  
- Eleição
